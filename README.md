# Namma-Santhe Ledger

Namma-Santhe Ledger is an Android application designed to manage ledgers for local markets (Santhe).

## Getting Started

This project is built using modern Android development practices, including:
- **Jetpack Compose** for UI
- **Kotlin**
- **Gradle Kotlin DSL**

### Prerequisites

- Android Studio Koala or newer
- JDK 17 or newer
- Android SDK 34 or newer

### Building

To build the project, open it in Android Studio or run:
```bash
./gradlew assembleDebug
```

## Features

- Digital ledger for market transactions
- User-friendly interface built with Jetpack Compose
- Modern Android architecture
